# Security Policy

If you find a security vulnerability or a device-safety issue in this module, please do not publish exploit details first.

Open a GitHub issue with a short, non-sensitive summary, or contact the maintainer privately through the GitHub profile associated with this repository.

Include:

- Module version.
- Device firmware or ROM build.
- Root solution and version.
- Reproduction steps.
- Relevant logs with personal data removed.

This project is maintained on a best-effort basis.
